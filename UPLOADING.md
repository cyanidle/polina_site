# Как загружать контент

## Комикс

```
comics/ru/Полный выплатец/
├── meta.json          ← необязательно
├── 0. Обложка.png
├── 1стр.png
├── 2стр.png
└── ...
```

`meta.json`:

```json
{
  "title": "Полный выплатец",
  "description": "О чём комикс.",
  "cover": "0. Обложка.png",
  "teaser": ["1стр.png", "10стр.png"],
  "characters": [
    { "name": "Guccio Belatti", "about": "Кто это" }
  ],
  "pages": {
    "3стр.png": { "comment": "Комментарий автора", "date": "2026-01-15" }
  }
}
```

Все поля необязательны. Без `meta.json` название = имя папки, обложка = первая страница по алфавиту/номеру.

Для английской версии — та же папка, но под `comics/en/`.

## Перевод текста в одном meta.json

Если картинки одни и те же, а разница только в тексте — `title`, `description`, `about` персонажа и `comment` страницы можно вместо строки задать объектом с языками. Сайт сам подставит нужный язык в зависимости от переключателя RU/EN:

```json
{
  "title": { "ru": "Полный выплатец", "en": "Paid in Full" },
  "description": { "ru": "О чём комикс.", "en": "What it's about." },
  "pages": {
    "3стр.png": {
      "comment": { "ru": "Комментарий автора", "en": "Author's comment" },
      "date": "2026-01-15"
    }
  }
}
```

## Комикс с главами

Если внутри папки комикса лежат подпапки — каждая подпапка становится отдельной главой:

```
comics/ru/Мой комикс/
├── meta.json
├── Глава 1/
│   ├── 1.png
│   └── 2.png
└── Глава 2/
    ├── 1.png
    └── 2.png
```

## Арт

```
arts/Закат.png
arts/Закат.txt   ← необязательное описание
```

Название = имя файла. Файл `.txt` с тем же именем — текст описания, который покажется рядом с работой.

## Формат страниц

Имена файлов сортируются "по-виндовски" (`2.png` раньше `10.png`), нумерация с ведущими нулями не нужна.
